# 📰 React News App

## 📌 About Project

React News App is a web application built using React.js that consumes the GNews REST API to fetch and display real-time news articles. The project demonstrates React Hooks, API integration, state management, and dynamic rendering of data.

## 🚀 Features

- Fetch latest news headlines from GNews API
- Get news on button click
- Refresh latest news anytime
- Display complete news details:
  - News image
  - News title
  - Description
  - Source name
  - Published date
- Read More option to open the original article website
- Loading indicator during API requests
- Clean and responsive user interface

## 🛠️ Technologies Used

- React.js
- React Hooks (`useState`)
- JavaScript (ES6+)
- Vite
- HTML5
- CSS3
- GNews REST API

## ⚙️ Installation and Setup

Clone the repository:

```bash
git clone https://github.com/Keerthi-111-spec/react-news-app.git
```

Navigate into the project folder:

```bash
cd react-news-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will run at:

```
http://localhost:5173/
```
