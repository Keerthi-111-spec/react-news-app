# 📰 React News App

A React-based news application that fetches the latest headlines using the GNews API. Users can get the latest news, refresh articles, view details like source and publishing time, and read the complete article from the original website.

## 🚀 Features

- Fetch latest news headlines
- Get news on button click
- Refresh news anytime
- Display:
  - News image
  - Title
  - Description
  - Source
  - Published date
- Read More option to open the original article
- Loading indicator during API requests
- Clean and responsive user interface

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- Vite
- HTML5
- CSS3
- GNews API

## ⚙️ Installation and Setup

Clone the repository:

```bash
git clone https://github.com/Keerthi-111-spec/react-news-app.git
```

Navigate to the project folder:

```bash
cd react-news-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
